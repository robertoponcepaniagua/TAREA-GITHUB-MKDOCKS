Probando markdown

Markdown es un lenguaje de marcado ligero que permite dar formato al texto (negrita, títulos, listas) utilizando símbolos simples en archivos de texto plano. Se usa principalmente para escribir de forma rápida y legible documentación técnica, artículos web, notas en apps como Obsidian o Notion, y para estructurar contenido en GitHub
