Introducción a HTML

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). No es un lenguaje de programación, sino un lenguaje que sirve para organizar el contenido de una web.

Estructura Básica

Cualquier archivo HTML empieza con estas etiquetas:

<!DOCTYPE html>
<html>
  <head>
    <title>Mi Tarea</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
  </body>
</html>

  • Títulos: Se usan etiquetas desde la h1 hasta la h6.

  • Párrafos: Se encierran en la etiqueta p.

  • Imágenes: Se usa la etiqueta img para mostrarlas.
